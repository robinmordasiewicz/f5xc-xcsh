    xcsh network_security apply¶

    Apply Network Security resources

    Synopsis¶

    xcsh network_security apply <resource-type> [name] [flags]

    Description¶

    Apply Network Security resources in F5 Distributed Cloud

    Examples¶

    xcsh network_security apply

    Common Usage Patterns¶

    Apply (create or replace) from file¶

    xcsh network_security apply <resource-type> -i resource.yaml
